EduBeatrix
==========

![](logo.jpg)

Fork of Beatrix, a low-rez music HTML5 game for LOWREZ jam 2014. https://github.com/cxong/Beatrix

How it works
----------------------
- Drag drums to build a pattern. Beats travel across the grid and trigger drums they collide with.
- The solution area shows the target pattern. Click PLAY to preview it.
- Each step shows a green or red marker for correctness. Match all steps to win.
- Click STOP to stop the preview loop. Use PAUSE to freeze the game. Adjust BPM with +/-.

Com funciona (valencià)
-----------------------
- Arrossega els tambors per crear un patró. Els beats es desplacen per la quadrícula i activen els tambors quan hi colpegen.
- L'àrea de solució mostra el patró objectiu. Fes clic en PLAY per escoltar-lo.
- Cada pas mostra un marcador verd o roig segons siga correcte. Completa tots els passos per guanyar.
- Fes clic en STOP per parar la repetició. PAUSE congela el joc. Ajusta els BPM amb +/-.

Como funciona (castellano)
--------------------------
- Arrastra los tambores para crear un patrón. Los beats se desplazan por la cuadrícula y activan los tambores al colisionar.
- El área de solución muestra el patrón objetivo. Pulsa PLAY para escucharlo.
- Cada paso muestra un marcador verde o rojo según sea correcto. Completa todos los pasos para ganar.
- Pulsa STOP para parar la repetición. PAUSE congela el juego. Ajusta los BPM con +/-.
